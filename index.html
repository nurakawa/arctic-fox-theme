about


Welcome to my website! I maintain this page to share my projects, blog posts, and interests. I am always open to conversations about new ideas and projects!

I studied Statistics at UC Berkeley, and I am fascinated with statistical machine learning and large-scale data analysis. I am a huge supporter of the R Project for Statistical Computing, and I am currently working with the U.C. Berkeley Department of Environmental Engineering on an R package for predictive modeling in Geosciences.


projects


  • Website: arnaout.ai

    (HTML, CSS, Ruby)

    I created the official website of the Arnaout Lab at UCSF (currently hosted here). The website uses the Minimal Mistakes Jekyll theme, and the code is open-source and available on Github .



  • Minority-Serving Institutions

    (R, Shiny, Web-scraping, Gradient-boosted trees, Lasso)

    This project was completed for the Reproducible and Collaborative Statistical Data Science course at U.C. Berkeley. The objective of this project was to use open-access data to determine universities that serve minority students. The project utilized data cleaning, visualization, and classification. The code and report is available on Github, as are two Shiny Apps (here) and (here).
